# git-batch-cloner

## Install

Using npm:

```console
npm install git-batch-cloner -g
```

Using yarn:

```console
yarn add git-batch-cloner -g
```

## Usage

```sh
gitlab-clone --url=https://gitlab-url --token="your person token" --output="./repo"
```

#### Using dotenv parameters

```sh
# .env file
URL=https://gitlab-url
TOKEN=your person token
```
